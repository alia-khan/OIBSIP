 To-Do Web Application

 Objective

A task management application for adding, editing, deleting, and completing daily tasks.

 Technologies Used

* HTML5
* CSS3
* JavaScript

 Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Task categories
* Search functionality
* Local storage support
* Responsive design

 How to Run

Open `index.html` in a browser.
