Login Authentication System

 Objective

A simple authentication system with registration, login, and a protected dashboard.

 Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage

 Features

* User registration
* Login functionality
* Password validation
* Password hashing
* Duplicate account prevention
* Remember Me option
* Dark mode toggle
* Show/Hide password feature
* Protected dashboard
* Logout functionality

 How to Run

Open `index.html` in your browser.
